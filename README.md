# COVID-Xray: анализ и очистка данных

Проект посвящён анализу и предобработке табличной части датасета **COVID-Xray** (метаданные к рентген-снимкам грудной клетки) с целью подготовки данных для последующего моделирования (например, классификации COVID vs non-COVID). 

---

## Описание проекта

В рамках проекта выполняются:

* загрузка и первичная очистка исходных данных;
* выявление проблем качества (пропуски, выбросы, дубликаты, грязные категории);
* нормализация и унификация признаков (возраст, даты, бинарные флаги, диагнозы);
* формирование финального очищенного датасета;
* базовый анализ и формулировка основных выводов и ограничений.

## Цель проекта

Разработать аналитическую систему для эпидемиологического мониторинга COVID-19 на основе метаданных рентгеновских снимков, используя стек PySpark.

## Используемый стек

* Язык: Python 3.x
* Основные библиотеки:
* pyspark — основная работа с данными (Spark DataFrame / SQL)
* pandas, numpy — вспомогательный анализ (опционально)
* matplotlib, seaborn — визуализации (опционально)
* jupyter — для запуска ноутбуков

### Зависимости для Spark

Для работы с PySpark требуется:

* установленный **Python 3.x**;
* установленный **Java (JDK/JRE)** и доступная команда `java` в PATH;
* Python-библиотека `pyspark` (ставится через `pip install pyspark` или `requirements.txt`).

Отдельно устанавливать полный Apache Spark-кластер не обязательно для локального анализа — достаточно `pyspark` и Java.
